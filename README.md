# OpenSCAD Course Project

## Постановка задачи

Подготовить материалы для курсовой работы по дисциплине «Программирование мехатронных систем» для варианта № 3.

Нужно было:
- получить 3D-модель детали;
- получить 3D-модель заготовки;
- собрать итоговый документ курсовой работы;
- использовать реальные скриншоты моделей из OpenSCAD в качестве иллюстраций.

## Краткое решение

Для проекта был подготовлен параметрический файл OpenSCAD с тремя режимами отображения:
- отдельная деталь;
- отдельная заготовка;
- деталь и заготовка рядом.

На его основе были сделаны скриншоты моделей, после чего они были использованы в итоговом документе курсовой работы `kursovaya_variant_3.docx`.

## Структура проекта

```text
OpenSCAD/
├── artifacts/
│   └── variant3_openscad.scad        # параметрическая модель варианта 3 для OpenSCAD
├── screenshots/
│   ├── detal.png                     # скриншот 3D-модели детали
│   ├── detal2.png                    # дополнительный скриншот модели
│   └── detals.png                    # скриншот заготовки / общего вида
├── kursovaya_variant_3.docx          # итоговая курсовая работа
├── LICENSE                           # лицензия репозитория
└── README.md                         # описание проекта
```
